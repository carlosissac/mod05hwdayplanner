Day Planner
How To Use:
  • Click on Left or Right arrows to shift through the calendar.
  • Select an empty slot, fill out the info and save.
  • Click on Time Format to switch between 24Hr. and AM/PM format.
  • Select Mock Events in order to load fake events in order to test functionality.
  • Click on Clear All to remove all events stored in local storage.
  • Click on an Existing Event in order to edit or delete.
  • Select an Expired Event in order delete.
  • UI will Autorefresh every 5 min.
  • Current hour will Autoscroll to the top everytime UI is autorefreshed.
  • Picture was included in order to confirm site responsiveness to different screen sizes.
  • Image has a link to the artist's social media page embedded.
